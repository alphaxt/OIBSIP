# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts the selling price of used cars using Machine Learning regression models. The dataset is taken from the CarDekho Vehicle Dataset available on Kaggle. The project includes data preprocessing, feature engineering, exploratory data analysis (EDA), model training, evaluation, and feature importance analysis.

---

## 🎯 Objective

Build a regression model that predicts the selling price of a used car based on features such as:

- Brand
- Car Age
- Present Price
- Kilometers Driven
- Fuel Type
- Transmission
- Selling Type
- Number of Owners

---
