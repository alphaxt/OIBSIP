# OIBSIP

## Unemployment Analysis with Python

### Objective

Analyze unemployment trends across Indian states and assess the impact of COVID-19.

### Technologies Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

### Dataset

Unemployment in India Dataset from Kaggle.

### Features

- Data cleaning
- Exploratory data analysis
- Time series analysis
- Correlation heatmap
- COVID-19 impact analysis

### Results

The analysis revealed significant regional differences in unemployment rates and demonstrated a sharp increase in unemployment during the COVID-19 pandemic.
